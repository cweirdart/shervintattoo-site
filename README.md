### The 30-second version

To make any change:
1. Go to **github.com/cweirdart/shervintattoo-site** in your browser
2. Press the **period key (`.`)** — that opens the full editor in your browser
3. Find what you want to change (search for `EDIT:` to jump between sections)
4. Save → commit → push (one button: "Commit & Push" in the Source Control panel)
5. Site updates at shervintattoo.com within 60 seconds

If something breaks, see "If something breaks" in ONBOARDING.md.
For anything bigger than a small edit, text c.weird at **info@cweird.com**.

## Quick technical facts

- **Stack:** Static HTML (no build step), Netlify hosting, GitHub for source

- **Form backend:** Netlify Forms (submissions email to shervinx510@gmail.com)
- **Deploy:** Auto-deploys on push to `main`
