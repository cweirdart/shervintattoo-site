# shervintattoo.com

Source for **Tattooing by Shervin** — live at https://shervintattoo.com.

Built and maintained by [c.weird](https://cweird.com).

---

## Quick technical facts

- **Stack:** Static HTML (no build step), Netlify hosting, GitHub for source
- **Spam protection:** Netlify honeypot + Cloudflare Turnstile + custom JS checks
- **Form backend:** Netlify Forms (booking + shop-order)
- **Deploy:** Auto-deploys on push to `main` (~60 seconds)
- **Domain:** shervintattoo.com (Namecheap, owned by Shervin)

## Editing the site

Site owner: editing instructions are in your private docs folder. Contact c.weird if you need access.
